# MariLink-CE
MariLink CE is a rewrite of MariLink in python, made without vibecoding almost everything.

## Setup:
1. get the things needed with `pip install discord.py aiohttp python-dotenv`
2. create .env and add in TOKEN=BOT.TOKEN_12345 with "BOT.TOKEN_12345" being your bot's token
3. edit bot.py's emoji section and upload all the images in /res except for img.png to a server or to the bot's emojis page. you can also leave them blank or add in your own, but i'd highly recommend not changing the reply and command ones.
4. run bot.py (python bot.py)
